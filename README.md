# AI量化交易学习项目 📈🤖

欢迎来到AI量化交易学习项目！这个项目专为Python初学者设计，帮助您从零开始学习Python编程和AI量化交易知识。

## 🎯 学习目标

通过这个项目，您将学会：
1. **Python基础编程** - 变量、函数、类、模块等
2. **数据分析技能** - pandas、numpy、matplotlib使用
3. **金融市场知识** - 股票、期货、加密货币等基础概念
4. **技术分析** - 各种技术指标的计算和应用
5. **量化策略开发** - 从简单到复杂的交易策略
6. **回测系统** - 如何验证策略的有效性
7. **机器学习应用** - AI在金融预测中的应用
8. **风险管理** - 如何控制交易风险

## 🚀 快速开始

### 1. 环境准备

```bash
# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 运行第一个示例

```bash
# 进入示例目录
cd examples

# 运行基础数据获取示例
python basic_data_fetch.py

# 运行简单移动平均策略
python simple_moving_average.py
```

## 💡 核心概念解释

### 量化交易是什么？
量化交易是使用数学模型和计算机程序来进行投资决策的方法。它的优势包括：
- 消除情绪干扰
- 提高执行效率
- 可以处理大量数据
- 策略可以回测验证

### AI在量化交易中的应用
- **价格预测**: 使用机器学习预测价格走势
- **风险管理**: AI模型评估投资组合风险
- **因子挖掘**: 发现新的投资因子
- **高频交易**: 毫秒级的交易决策

## 📊 示例策略说明

### 1. 简单移动平均策略
当短期均线上穿长期均线时买入，下穿时卖出。
- 文件位置: `examples/simple_moving_average.py`
- 适合初学者理解趋势跟踪概念

### 2. RSI均值回归策略
使用相对强弱指数(RSI)识别超买超卖机会。
- 文件位置: `examples/rsi_strategy.py`
- 学习技术指标的计算和应用

### 3. 机器学习预测策略
使用随机森林预测下一日收盘价方向。
- 文件位置: `examples/ml_prediction_strategy.py`
- 理解特征工程和模型训练
