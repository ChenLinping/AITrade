# AITrade - AI量化交易系统

这是一个AI驱动的量化交易系统，旨在帮助用户进行智能投资决策。

## 项目特性

- 智能交易策略
- 风险管理系统
- 回测功能
- 多种技术指标分析

## 文件结构

- `src/` - 源代码目录
- `examples/` - 示例代码
- `tutorials/` - 教程文件
- `src/main.py` - 主程序入口点

## 快速开始

安装依赖：
```
pip install -r requirements.txt
```

运行主程序：
```
python src/main.py
```

## 当前功能

当前版本包含一个红利-科技周频调仓模拟器，具有以下功能：
- 手动输入周涨跌幅
- 提供调仓建议和金额
- 内部模拟仓位管理
- 自动保存历史记录
- 图形化界面操作
